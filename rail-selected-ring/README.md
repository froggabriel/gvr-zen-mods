# [GVR] Rail Selected Ring

**Version:** 1.0.0

Collapsed sidebar rail: subtle tile backgrounds on all tabs, with a selected-tab inset ring. Container tabs (`usercontextid`) use `--identity-icon-color` for the ring so it matches the stripe.

Optional companion for `tab-containers` and expand-on-hover. Disable independently if you prefer Zen’s default selected styling.

## Install

```bash
python3 install.py rail-selected-ring
```

Restart Zen Browser to apply.
