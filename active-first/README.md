# [GVR] Active First

**Version:** 1.0.0

In rail mode, active tabs float to the top and inactive tabs are dimmed below. In expanded mode, inactive tabs are hidden.

Companion for `zen-sidebar-expand-on-hover`.

## Preferences

- **Hide inactive tabs in rail** (default: on) — inactive unpinned tabs are hidden in the collapsed sidebar.

## Install

From the repo root:

```bash
python3 install.py active-first
```

Restart Zen Browser to apply.
